Run the following code:

## Data Source

Saudi Ministry of Justice, which can be found here: <a href="https://laws.moj.gov.sa/ar/legislations-regulations/">link</a>.<br>
To download all the PDF files into your system run `scrape_pdfs.py` which will scrape the website mentioned above and download all the files, the whole process will take around <strong style="color:rgb(239, 90, 90)">8 minutes, make sure to have Google Chrome installed as well</strong>.

```bash
$ python src/Scrape/scrape_pdfs.py
```
